# 📊 Sales Performance Matrix Dashboard

An interactive Sales Performance Dashboard built using **Power BI** to analyze business performance across quarters, states, customers, and product categories.

---

## 🚀 Project Overview

This dashboard provides a comprehensive analysis of sales performance, enabling dynamic filtering and insight generation based on:

- Quarter-wise performance
- State-level analysis
- Customer contribution
- Product category & sub-category analysis
- Payment mode distribution
- Monthly profit trends

The objective of this project is to transform raw sales data into meaningful business insights using data modeling, DAX, and interactive visualization techniques.

---

## 📌 Key KPIs

- 💰 Total Sales
- 📈 Total Profit
- 📦 Total Quantity Sold
- 🧾 Average Order Value (AOV)

These KPIs provide an executive-level summary of overall business performance.

---

## 📊 Dashboard Features

### 🔹 Quarter-Based Slicer
Allows dynamic analysis of Q1, Q2, Q3, and Q4 performance.

### 🔹 State-Based Slicer
Filters the dashboard to analyze regional sales insights.

### 🔹 Monthly Profit Analysis
Identifies positive and negative profit trends month-wise.

### 🔹 Top 3 Customers
Highlights highest spending customers.

### 🔹 Sub-Category Performance
Shows profit contribution by product sub-category.

### 🔹 Payment Mode Distribution
Analyzes customer payment behavior (COD, UPI, EMI, Credit Card, Debit Card).

---

## 🛠 Tools & Technologies Used

- Power BI Desktop
- Power Query (Data Cleaning & Transformation)
- DAX (Calculated Columns & Measures)
- Data Modeling (Relationships)
- Interactive Visual Design

---

## 📂 Dataset Information

The dataset consists of two tables:

- `orders`
- `details`

Data was cleaned, transformed, and enhanced with additional calculated columns before building the dashboard.
